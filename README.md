**X (Twitter)**
---
X (бывший Twitter) - социальная сеть для публичного обмена короткими сообщениями в реальном времени через специализированную ленту рекомендаций. Пользователи публикуют и взаимодействуют с сообщениями (твитами).

**MVP**

1. Лента твитов (постов) пользователей (лента "Для вас", без подписок)
2. Публикация твитов (постов), ответы/репосты/лайки

**Целевая аудитория (обновлено в январе 2026 года)** [1]

- В настоящее время в Twitter насчитывается около **600 с лишним миллионов** ежемесячных активных пользователей (MAU).
- Более **235 с лишним миллионов** человек ежедневно пользуются X/Twitter. (DAU)
